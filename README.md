# antsTransformTests
Tests for handling and applying ANTs transforms

Run `registration.sh` first to generate transforms.


## Composite transform tests

This tests for consistency with different transform formats. Run
`test_composite_transforms.sh`.


## Input image type tests

This tests for consistency with time series and multi-channel images.

Run `test_input_image_types.sh`.

More detailed tests on tensor and vector input are in
[antsDTOrientationTests](https://github.com/cookpa/antsDTOrientationTests/tree/main).
